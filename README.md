# 鉴渣助手

一个微信小程序原型，用于生成亲密关系风险识别报告。产品采用三页流程：启动页、信息输入与测试页、测评报告页。

## 当前结构

- 启动页：黑白极简文档风，突出产品名、多维渣值测试和中西结合卖点。
- 测试页：录入姓名、生日（公历，最晚 2011-12-31）、出生时间段、性别，然后完成 30 道关系信号题。
- 报告页：输出综合渣值、风险等级、整体画像、六大维度详情、中西结合详情、风险解释、行动建议，并支持生成报告、一键下载和分享好友。
- 下载报告图：使用动态高度长图，六大维度以六边形图展示，中西结合单独成块上下排列，保留核心四柱与星盘分析。

## 六大维度

测试题统一折算到以下结果维度，所有报告判断都来自对应题目，不做无依据推断。

| 维度 | 关注点 |
| --- | --- |
| 情感忠诚风险 | 关系透明度、是否愿意给确定性、是否隐瞒关键事实 |
| 出轨/多线风险 | 暧昧边界、隐藏社交信息、多线历史和越界调整意愿 |
| 亲密处理风险 | 冲突沟通、冷处理、断联、修复能力 |
| 价值观与尊重风险 | 平等尊重、边界感、双标、情绪施压和共情能力 |
| 金钱与利益风险 | 借钱转账、消费绑定、投资项目、经济责任双标 |
| 责任感与承诺风险 | 承诺兑现、甩锅、长期责任和实际改变 |

## 中西结合

- 东方部分基于公历出生年月日时做四柱简析，包含年柱、月柱、日柱、时柱、日主、五行、桃花星、夫妻宫/日支、十神提示。
- 西方部分基于生日与出生时间做太阳星座、月亮星座估算、四元素和三模式分析。
- 中西结合用于解释关系风格和潜在倾向，不单独替代现实行为判断。

## 风险等级

| 综合渣值 | 风险等级 |
| ---: | --- |
| 0-20 | 较低风险 |
| 21-45 | 低风险 |
| 46-70 | 中风险 |
| 71-85 | 高风险 |
| 86-100 | 极高风险 |

## 使用方式

1. 用微信开发者工具打开本目录。
2. 以游客 AppID 运行即可预览。
3. 从启动页点击“开始测试”，完成信息录入和 30 道题。
4. 报告页底部点击“生成报告”，再点击“一键下载”保存到相册，也可以点击“分享好友”。
