# Requirements

## Goal

Build and maintain the "鉴渣助手" WeChat mini program for relationship risk assessment reports.

## Functional Requirements

- Provide a start page, test flow, analysis generation page, and report page.
- Generate a relationship risk score, risk level, relationship risk type, dimension summaries, and action suggestions.
- Support report image generation, saving, and sharing.

## Non-Functional Requirements

- Do not commit secrets, environment files, or local private configuration.
- Keep the mini program package small enough for WeChat upload limits.
- Preserve the existing black-and-white document-style visual direction unless a redesign is explicitly requested.
